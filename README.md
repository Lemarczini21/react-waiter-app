# WaiterApp
Application for waiters to handle orders written in React.
> Live demo https://react-waiter-app.lemarczini21.repl.co/.


## Technologies Used
- React
- Redux
- React Bootstrap


## Features
- Render tables from Redux store
- Edit table
- WEB API
